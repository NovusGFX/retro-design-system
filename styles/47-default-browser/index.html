About this page

You are reading an example of the Default Browser aesthetic. It is not nostalgia. It is not brutalism. It is the rendering engine, exposed. A reminder that HTML was already a complete UI before CSS existed.

  • No custom fonts
  • No colors except link defaults
  • No animations
  • No JavaScript
  • Just semantic markup, in the order it appears
