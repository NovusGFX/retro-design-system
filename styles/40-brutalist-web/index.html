Manifesto

  1. A website's performance is more important than its aesthetics.
  2. Readable text is a feature.
  3. Links should look like links. Buttons should look like buttons. Tables like tables.
  4. The back button should work.
  5. If the CSS fails to load, the page should still be useful.
